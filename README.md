### Website

Just using this repo to also prototype some website ideas

[http://stage.stopthespies.org/](http://stage.stopthespies.org/)

##### To setup dev environment

```
rvm use 2.1.2		# probably works with other ruby versions
bundle install
npm install -g bower
bower install
```

##### To develop

```
jekyll serve --watch
```

##### To build and delpoy

```
rake site:publish
```
